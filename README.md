# Web Rubik Cube

## Description

此项目使用 Three.js 完成。

当前预计功能有：

- [x] 魔方的 3D 显示
- [ ] 对魔方进行基本操作
- [ ] 导入／导出魔方状态

## Usage

- 在控制台使用以下代码可使魔方随机打乱并还原。

```javascript
const commandStr = randomCube(20);

executeCommands(commandStr);

reverseCommands(commandStr);
```

- 使用键盘按键「←」「→」可使魔方整体旋转（实际上是摄像机在其轨道上移动）。

## Bug

- 当 `duration` 的值过小时，魔方会出现当前旋转动画尚未完成便进行下一次旋转，导致魔方错位。目前推荐的最小值是 300。

	解决办法：定期对魔方进行检查，修复魔方。
